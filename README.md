# About this repository

This repository is used to manage articles published on https://zenn.dev/yamatech via GitHub.

It contains the Markdown sources under `articles/` and `books/`, along with `images/` referenced from them, and is kept in sync with Zenn via the GitHub integration.


## Zenn CLI
Use the Zenn CLI to create and preview articles locally before publishing.
* [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)

## Create New Article
Use Makefile commands.
1. make new
2. make preview
3. git push to zenn repository
